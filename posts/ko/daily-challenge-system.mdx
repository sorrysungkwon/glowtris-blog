글로우트리스에는 각 사용자의 현지 시간대 기준 자정에 "초기화"되는 일일 리더보드가 있습니다. 듀오링고의 주간 리그와 같은 개념입니다. 매일 새로운 느낌을 주어 다시 확인하고 싶게 만들죠.

그런데 실제로는 데이터가 삭제되지 않습니다.

## 단순한 접근법과 왜 안 되는지

가장 직관적인 구현은 UTC 자정에 리더보드를 지우는 cron 작업을 실행하는 것입니다. 간단해 보이죠.

하지만 그렇게 하면 서울 플레이어는 오전 9시에 초기화를 보고, 뉴욕 플레이어는 전날 오후 7시에 봅니다. "자정 초기화" 느낌이 사라지고, 실제 초기화 시각은 그 누구에게도 자정이 아닌 임의의 UTC 시점이 됩니다.

또 다른 단순한 접근법은 모든 시간대마다 cron을 돌리는 것입니다. 꼼꼼히 하려면 38개의 IANA 시간대가 필요하고, 각각 별도의 리더보드가 필요합니다. 가능하긴 하지만 지저분합니다.

## 실제로 작동하는 것: 날짜별 버킷

리더보드를 "초기화"한다는 개념 대신, 다르게 생각해봅시다. 날마다 별도의 리더보드 키를 갖는 것입니다.

```
glowtris-daily-2026-06-04  ← 어제 보드
glowtris-daily-2026-06-05  ← UTC-5에 있는 사람의 오늘 보드
glowtris-daily-2026-06-06  ← UTC+9에 있는 사람의 오늘 보드
```

클라이언트는 매 요청마다 현지 날짜를 보냅니다:

```javascript
const d = new Date()
const localDate = `${d.getFullYear()}-${String(d.getMonth()+1).padStart(2,'0')}-${String(d.getDate()).padStart(2,'0')}`

fetch(`/api/leaderboard?date=${localDate}`)
```

서버는 그 날짜를 리더보드 키로 사용합니다. 현지 자정이 지나면 클라이언트의 `localDate`가 바뀌고, 비어 있는 새 버킷을 보고 거기에 점수를 올리게 됩니다. 전날 버킷은 Redis에 그대로 남아 있지만 더 이상 표시되지 않을 뿐입니다.

## 보안 고려사항

당연히 떠오르는 질문: 예전 날짜를 보내서 어제 리더보드에 점수를 올릴 수 있지 않을까?

이론적으로는 가능합니다. 실제로는 간단한 서버 측 검사를 추가했습니다:

```javascript
function safeClientDate(raw) {
  const utc = new Date().toISOString().slice(0, 10)
  if (!raw || !/^\d{4}-\d{2}-\d{2}$/.test(raw)) return utc
  // UTC 오늘과 1일 이상 차이나는 날짜는 거부
  return Math.abs(new Date(raw) - new Date(utc)) <= 86400000 ? raw : utc
}
```

최대 합법적인 시간대 오프셋은 UTC+14(키리바시 일부)로, 클라이언트 현지 날짜와 UTC 오늘의 최대 차이가 정확히 하루입니다. 따라서 이 검사는 시간대 기반의 모든 합법적인 날짜를 허용하면서 수상한 것은 거부합니다.

## TTL 계산

오래된 Redis 키는 결국 만료되어야 합니다. 원래 일일 TTL은 26시간이었는데, UTC 기준 날짜에는 충분한 여유였습니다.

시간대 인식 날짜를 사용하면, `2026-06-06` 키가 UTC+14 사용자에 의해 6월 5일 UTC 10:00에 처음 쓰이고, UTC-12 사용자에 의해 6월 8일 UTC 12:00에 마지막으로 쓰일 수 있습니다. 약 50시간입니다.

그래서 TTL을 52시간으로 설정했습니다. 모든 시간대의 모든 사람이 하루를 마칠 때까지 데이터가 남아 있다가, 조용히 만료됩니다.

---

그래서 결론은 간단합니다. 서버 하나가 시간을 봅니다. 끝.

당신이 서울에서 자정 직전에 플레이합니다. 자정 되면 리더보드 바뀝니다. 비행기 타고 샌프란에 내립니다(현지 시간으로 12시간 뒤). 근데 서버 입장에선 2시간 정도만 지났어요. 리더보드는 그대로 있고, 스트릭도 끊기지 않았고, 당신은 여전히 전 세계 모든 사람과 같은 "날"에 있습니다.

그게 다야. 서버는 샌프란이 뭐 시간인지 신경 안 써요. 당신이 뭐 시간인지도 모르고. 그냥 자기가 뭐 시간인지만 알아요.
